# Dots and Boxes

## How to run
1. Clone the repository
```
git clone https://github.com/ilcho02/dots_and_boxes
```
2. Navigate to dots_and_boxes
```
cd dots_and_boxes
```
3. Create a virtual environment
```
python -m venv .venv
```
4. Install the dependencies
```
pip install -r requirements.txt
```
5. Navigate to src
```
cd src
```
6. Run the game
```
python main.py
```
